# Chrome-extension Template (built with TypeScript + React + Semantic UI)

quickly build chrome extension using TypeScript and React

## Setup(npm)
```
$ npm install
```

### Development
```
$ npm run dev
```
```
$ npm run prod
```

### Build
```
$ npm run build
```

## Setup(docker)
```
make docker-build
```

### Development
```
$ make docker-dev
```
```
$ make docker-prod
```

### shell
```
$ make docker-shell
```
